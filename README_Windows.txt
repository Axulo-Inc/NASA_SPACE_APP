CLIMAGUARD AI - WINDOWS SETUP

PREREQUISITES:
1. Install Python from python.org (check "Add Python to PATH")
2. Restart computer after Python installation

QUICK START:
1. Double-click "setup.bat" (installs dependencies)
2. Double-click "run_demo.bat" (starts the app)
3. Open browser to: http://localhost:8080

DEMO LOCATIONS:
- Johannesburg: -26.2041,28.0473
- Cape Town: -33.9249,18.4241
- Nairobi: -1.2921,36.8219

TROUBLESHOOTING:
- If Python not found: Install from python.org
- If port busy: We'll use port 8081 instead
- If any issues: Use manual commands in Command Prompt
